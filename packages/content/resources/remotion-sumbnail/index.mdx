## Video Thumbnail Generator

Generate YouTube thumbnails programmatically with Remotion's `renderStill` command. The composition renders a 1280×720 frame with a bold gradient background, oversized title text, a category badge, and a placeholder slot for a face or product image.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 × 720 |
| FPS | 30 |
| Duration | 1 s (30 frames — effectively a still) |

### Render as still

```bash
npx remotion still remotion-sumbnail thumbnail.png --props='{"title":"My Video","tag":"Tutorial"}'
```

### Props

| Prop | Default | Description |
|---|---|---|
| `title` | `"Video Title"` | Main headline text |
| `tag` | `"Tutorial"` | Badge label (top-left) |
| `bg1` | `"#0f0c29"` | Gradient start color |
| `bg2` | `"#302b63"` | Gradient end color |
