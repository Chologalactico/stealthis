## Logo Reveal Animation

A cinematic 3-second logo reveal composition. The logo scales up from 0 using a spring with overshoot, while a radial gradient burst expands behind it. The tagline fades in with a small upward translate once the logo settles.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 × 720 |
| FPS | 30 |
| Duration | 3 s (90 frames) |

### Customization

Replace the `LOGO_TEXT`, `TAGLINE`, and `BRAND_COLOR` constants. To use an actual image instead of text, swap the `<div>` logo element for a Remotion `<Img>` component pointing to a `staticFile`.
