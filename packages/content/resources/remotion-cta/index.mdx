## Call-to-Action Animation

A 4-second animated CTA composition designed for use as a video overlay or standalone marketing clip. Features a sliding headline, a spring-animated button, a shimmer effect sweep, and a bouncing arrow indicator.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 × 720 |
| FPS | 30 |
| Duration | 4 s (120 frames) |

### Timeline

| Time | Action |
|---|---|
| 0–0.5 s | Headline slides in |
| 0.5–1 s | Subtext fades in |
| 1–1.5 s | Button springs up |
| 1.5–4 s | Arrow bounces; shimmer sweeps button |
