## Animated Number Counter Video

A 4-second Remotion composition where a large number counts up from 0 to its target value. Uses `interpolate` with an `easeOut` curve so it slows down as it approaches the final value. Supports prefix (`$`, `+`) and suffix (`K`, `%`, `M`) labels.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 × 720 |
| FPS | 30 |
| Duration | 4 s (120 frames) |

### Props

| Prop | Default | Description |
|---|---|---|
| `target` | `10000` | Final number value |
| `prefix` | `""` | Text before the number (e.g. `"$"`) |
| `suffix` | `"+"` | Text after the number (e.g. `"K"`) |
| `label` | `"Total Users"` | Caption below the number |
| `color` | `"#6366f1"` | Accent color |
