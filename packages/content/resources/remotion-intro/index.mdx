## YouTube Intro Animation

A 5-second YouTube channel intro built with Remotion. Animates the channel name sliding in from the left, the subtitle fading up, and a colored accent bar growing across the screen.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 × 720 |
| FPS | 30 |
| Duration | 5 s (150 frames) |

### Customization

Replace the `CHANNEL_NAME`, `TAGLINE`, and `BRAND_COLOR` constants at the top of the file to match your brand.

### How to register

```tsx
import { Composition } from "remotion";
import { YoutubeIntro } from "./remotion-intro";

export const Root: React.FC = () => (
  <Composition
    id="YoutubeIntro"
    component={YoutubeIntro}
    durationInFrames={150}
    fps={30}
    width={1280}
    height={720}
  />
);
```
